### Testing Template Development for Grav
#### Test Project – Why Agency

* **Date:** February 2021
* **Technologies:** PHP, Twig
* **Duration:** 1 day

{{gallery:0}}

As part of a client project, we needed a CMS that was lighter and more eco-friendly than WordPress. We therefore explored [Grav](https://getgrav.org/), an open-source CMS built on Symfony, using a *flat-file* system and Markdown for content.

Since Grav offers only a limited number of ready-to-use themes, I suggested creating a custom theme from a static HTML template: the [Bootstrap “Creative” theme](https://startbootstrap.com/theme/creative).

This project allowed me to quickly learn **Twig** and the process of building Grav themes. I analyzed the HTML structure to identify reusable elements, integrated them with Twig, and customized the back-office to simplify content management.

Within a single day, the theme was fully functional. This successful test expanded our internal skills and opened the way to building custom Grav themes better suited to client needs.
